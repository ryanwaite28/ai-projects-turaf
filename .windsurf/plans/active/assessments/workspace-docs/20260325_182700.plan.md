```markdown
# Common Issues and Solutions

## Infrastructure

### Terraform State Lock
**Problem**: Terraform state is locked
**Solution**: `terraform force-unlock <lock-id>`

### ECS Service Won't Start
**Problem**: ECS tasks failing to start
**Solutions**:
1. Check Docker image exists in ECR
2. Verify task definition CPU/memory limits
3. Check CloudWatch logs for errors

## CI/CD

### GitHub Actions OIDC Failure
**Problem**: Cannot assume AWS role
**Solutions**:
1. Verify OIDC provider exists in AWS account
2. Check IAM role trust policy
3. Verify branch name matches trust policy condition

## Database

### Connection Refused
**Problem**: Cannot connect to RDS
**Solutions**:
1. Verify security group allows traffic
2. Check RDS instance is running
3. Verify connection string and credentials

## More Issues

See service-specific troubleshooting in each service's README.
```
