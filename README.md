## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/ryanwaite28/ai-projects-turaf.git
cd ai-projects-turaf

# Start local environment (Docker required)
docker-compose up -d

# Verify services
docker-compose ps

# Access frontend
open http://localhost:4200
```

For detailed setup instructions, see **[PROJECT.md § 22a: Local Development Setup](PROJECT.md#22a-local-development-setup)**.
